FAQ

If you need more information please send a support ticket in the discord

    Questions

      Q: Say for example I captain Rank B, can I go back and fly any Rank C aircraft?

      A: Yes, you can,

      Q: What if I don't own an aircraft that is supported in SkyCab?

      A: You will not be able to participate on any SkyCab flights.

      Q: What if the livery file doesn't work on the aircraft I own?

      A: This is completely fine. Aslong if the plane is white it is fine.

      Q: What simulators do you currently support?

      A: We mainly support Prepar3D, however if the same aircraft are compatable we will accept a PIREP from another simulator such as X-Plane 11.

      *However we do not accept a Pirep filed from FlightGear nor do we accept Infnite Flight or any other mobile 'simulator'.

      Q: What if the livery file doesn't work on the aircraft I own?

      A: This is completely fine. Aslong if the plane is white it is fine.

      Q: Do I have to use the correct flight numbering for the flight?

      A: We would prefare if you did, however if your flight number is already in use you can make up your own.

      Q: Do I have to be connected to a Virtual Air Traffic control network and if so, which one?

      A: We don't require you to be connected to a VATC network, but we do support VATSIM, IVAO and Pilots Edge (Only if you have a subscription) We would recommend you to fly on VATSIM.

    If you have any more questions then please submit a support ticket on Discord.
